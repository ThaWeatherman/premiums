{% extends "base.html" %} {% block content %} The provided API is simple to use. There are 4 methods, each of which returns JSON. All are POST requests.
/api/to_grams converts grams to troy ounces. Submit your data with the parameter "grams" set to the amount of grams.
/api/to_ounces converts troy ounces to grams. Submit your data with the parameter "oz" set to the amount of troy ounces.
/api/over takes "spot" (spot price), "price" (price of the item), and "weight" (weight in troy ounces NOT GRAMS) and returns the $ amount over spot of the item.
/api/premium takes "spot" (spot price), "price" (price of the item), and "weight" (weight in troy ounces NOT GRAMS) and returns the percentage over spot of the item.
Please try not to overload the server with your requests. {% endblock %}
